# "对话即探索" —— 让AI交互像与一位了解你的老友聊天，自然、有温度、有记忆。

## 主页面是用户与 AI Agent 交互的聊天界面
- 聊天框上方给出一些快捷指令，比如"总结上文"、"换一个说法"、"深入聊聊"；检测到粘贴大段文字时自动询问"需要总结吗？"
- 聊天框空态时显示情境提示：根据时间、历史习惯动态变化（如早晨显示"今日计划？"，深夜显示"还在思考什么？"）
- 显示会话热度图，最近活跃的会话带有温度色调标识
- 打开应用时，AI 会结合近期用户记忆，生成主动问候词
- 跨会话记忆：AI能自然引用其他会话的相关内容（"你之前在'工作项目'里提到的方案...")
- 支持对话分支：对某条回复不满意？点击"换个思路"，生成平行对话线，询问用户更喜欢哪个回复

## 第二个页面是知识库页面
- 展示 AI 自动生成或者用户指定的记忆，记忆列表允许用户编辑与删除；每个记忆显示 AI 对其准确性的把握程度，长期未使用的记忆自动归档，但可随时唤醒
- 当新信息与旧记忆矛盾时（"你之前说讨厌香菜，现在...？"），主动提示核实
- 拖拽两个相关记忆，AI建议是否合并

## 第三个页面是个人设置页面
- 记录 token 消耗量（展示时间轴），不是简单数字，而是"相当于阅读3本小说"、"约等于从地球到月球的文字距离"


## 人性化细节

| 场景                       | 设计响应                                                               |
| -------------------------- | ---------------------------------------------------------------------- |
| 用户等待输出               | 显示"AI正在思考..."时附带趣味小知识或进度诗意化（"正在连接神经元..."） |
| 成就时刻                   | 完成复杂任务后，界面撒下微粒子动画庆祝                                 |
| 用户点入未实现功能的界面时 | 展示提示词 —— 这里是一片荒芜，去别处看看吧                             |
