# app_gui

## 为什么 Session 不可变

| 方面     | 不可变 Session（当前）        | 可变 Session             |
| :------- | :---------------------------- | :----------------------- |
| Riverpod | 响应式	state = 触发重建，可靠 | 字段直接改不触发 UI 重建 |
| 调试     | 每次变更都是新对象，可追踪    | 原地修改难以溯源         |
| 并发安全 | 无竞态                        | 需要加锁                 |
| 测试     | 容易断言前后状态              | 需要快照                 |
